# Release notes

What's new in FlowPlanner. Updates are listed newest first.

> [!quote] We use our own board
> FlowPlanner's feature roadmap is run on FlowPlanner. See what's coming next at **[flowplanner.flowplanner.io](https://flowplanner.flowplanner.io)**.

---

## 2026

### May 2026

**Theme preference sync across domains**
Your dark/light mode preference is now saved to your account. It applies on both the main site and your board subdomain, and persists when you sign in on a new device.

**Live board demo link**
The marketing landing page now links to a live FlowPlanner board so prospective users can see the product in action before signing up.

**OrgBoard toggle update**
The Feedback/Roadmap toggle in the board header has been reordered (Roadmap first) and is now centered in the header for a cleaner layout.

---

### April 2026

**Initial public launch**
FlowPlanner is live. Core features available at launch:
- Public feedback boards at `yourname.flowplanner.io`
- Voting, categories, and statuses
- Public roadmap
- Admin panel with full feedback triage
- Team member invitations
- Account system with email verification and magic links
